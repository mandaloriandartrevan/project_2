      Клиент-серверное приложение для мониторинга рабочей активности

Объяснение
   Простое приложение для отображения текущей рабочей активности всех работодателей в организации

Клиент (windows) - c/c++
   Автоматически запускается при входе пользователя в систему и работает в фоновом режиме.
   Взаимодействует с сервером по любому протоколу.
      Вы не можете использовать сторонние библиотеки,такие как boost и другие,
   и вы не можете использовать фреймворки, такие как Qt и другие. 

   Сервер - desktop приложение или веб-интерфейс - на любом языке программирования.  
   Перечислите все подключенные клиенты - домен/компьютер/ip/пользователь.
   Показывать время последней активности клиента.
   Возможность получения скриншота с рабочего стола клиента.

   Реализация:
   Сервер и Клиент были написаны на языке программирования Python,а логика работы самого сервера
была реализована на языке программирования Javascript.

   Алгоритм запуска данного программного продукта:
Чтобы запустить данный программный продукт сначала нужно:
 - Запустить файл Service_3.py.
 - Логика работы самого сервера реализована в файле server.js.
 - Затем запустить Client_2.py.
   *Также присутсутствует веб интерфейс в файле index_3.html
